## 功能特性

- **📊 实时额度看板**：同时展示 3-4 个模型的 **5h / 周 / 月** 三层额度
- **⏳ 滑动窗口倒计时**：精确计算 Coding Plan 的 **5h 滑动窗口** 释放时间
- **📈 占比可视化**：每个模型卡片内嵌三层进度条，一眼看出额度消耗比例
- **💡 RGB 状态指示灯**：1-3 颗 WS2812B 灯珠，颜色和规则完全自定义
- **🔋 低功耗常显**：墨水屏断电保持画面，Deep Sleep 待机功耗 < 10μA
- **🌐 双模式部署**：设备直连（Standalone）或网关代理（Gateway）两种架构

## 快速开始

### 1. 硬件接线

```
ESP32-S3          2.9寸墨水屏
-----------       -----------
3.3V    ───────── VCC
GND     ───────── GND
GPIO 18 ───────── SCK (CLK)
GPIO 23 ───────── DIN (MOSI)
GPIO 5  ───────── CS
GPIO 17 ───────── DC
GPIO 16 ───────── RST
GPIO 4  ───────── BUSY

ESP32-S3          WS2812B
-----------       -----------
GPIO 15 ───────── DIN (数据输入)
3.3V    ───────── VCC
GND     ───────── GND
```

### 2. 软件环境

```bash
# 安装 MicroPython 固件（ESP32-S3）
esptool.py erase_flash
esptool.py write_flash -z 0x0 ESP32_GENERIC_S3-20240105-v1.22.1.bin

# 部署项目代码
mpremote connect /dev/tty.usbserial-* fs cp src/ :
mpremote connect /dev/tty.usbserial-* run src/main.py
```

### 3. 配置 API

在设备或网关服务器上创建 `config.json`：

```json
{
  "wifi": {
    "ssid": "YourWiFi",
    "password": "YourPassword"

  "platforms": [

      "api_key": "sk-sp-xxx",
      "base_url": "https://coding.dashscope.aliyuncs.com"
    },
    {
      "name": "GLM",
      "type": "zhipu",
      "api_key": "xxx"
    }
  ],
  "led": {
    "pin": 15,
```
